Repositório contendo os exercícios, exemplos e projetos desenvolvidos durante as aulas da disciplina de Programação.

## Conteúdo das Aulas

### Aula 02 - Introdução à Lógica de Programação
**Conteúdos abordados:**
- Tipos de dados
- Variáveis e constantes
- Operadores
- Entrada e saída de dados
- Primeiros algoritmos

---

### Aula 03 - Git e GitHub
**Conteúdos abordados:**
- Conceitos de Git e GitHub
- Criação de repositórios
- Commits
- Branches
- Merge
- Principais comandos de versionamento

---

### Aula 04 - Estruturas de Seleção
**Conteúdos abordados:**
- `if`
- `else if`
- `switch`
- Operadores relacionais e lógicos
- Tomada de decisão

---

### Aula 05 - Estruturas de Repetição
**Conteúdos abordados:**
- `for`
- `while`
- `do while`
- Controle de laços
- Repetição de instruções

---

### Aula 06 - Funções
**Conteúdos abordados:**
- Criação de funções
- Parâmetros
- Retorno de valores
- Organização e reutilização de código

---

### Aula 07 - Vetores e Matrizes
**Conteúdos abordados:**
- Vetores
- Matrizes
- Percorrendo estruturas
- Manipulação de dados

---

### Aula 08 - Structs
**Conteúdos abordados:**
- Estruturas (`struct`)
- Organização de dados
- Agrupamento de diferentes tipos de informações

---

### Aula 09 - Listas e Dicionários
**Conteúdos abordados:**
- Listas e vetores dinâmicos
- Manipulação de elementos 
- Dicionários 
- Inferência de tipos com `var` (C#) e `auto` (C++)

---

## 🛠️ Linguagens utilizadas

- C#
- C++
